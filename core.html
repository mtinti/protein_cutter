digest

 digest (sequence:str, protein_id:str, enzyme:str='trypsin',
         missed_cleavages:int=1, charge_states:list=[1, 2, 3],
         mass_range:tuple=(800.0, 4000.0), min_pep_length:int=5,
         max_pep_length:int=35, sort_by_mass:bool=False)

Digest a protein and add flanking amino acids for each peptide.

Type Default Details
sequence str Protein sequence to digest
protein_id str Protein identifier (for the DataFrame)
enzyme str trypsin Enzyme name (default: ‘trypsin’)
missed_cleavages int 1 Number of allowed missed cleavages (default: 1)
charge_states list [1, 2, 3] Charge states for m/z calculation (default: [1, 2, 3])
mass_range tuple (800.0, 4000.0) (min, max) monoisotopic mass filter in Da (default: (800.0, 4000.0))
min_pep_length int 5 Minimum peptide length to retain (default: 5)
max_pep_length int 35 Maximum peptide length to retain (default: 35)
sort_by_mass bool False Sort output by monoisotopic mass (default: False)
Returns DataFrame DataFrame with the following columns:
- start_index : int - Start position in protein sequence
- end_index : int - End position in protein sequence
- pep_seq : str - Peptide sequence
- protein_id : str - Protein identifier
- pep_length : int - Peptide length
- prev_aa : str - Previous amino acid (or ‘-’ at N-terminus)
- next_aa : str - Next amino acid (or ‘-’ at C-terminus)
- extended_seq : str - Sequence with flanking AAs
- rep_extended_seq : str - Extended sequence with parentheses
- mass_mono : float - Monoisotopic mass (Da)
- mass_avg : float - Average mass (Da)
- mz_{z} : float - m/z for each charge state in charge_states

digest_to_set

 digest_to_set (sequence:str, enzyme:str='trypsin',
                missed_cleavages:int=0,
                mass_range:tuple[float,float]=(800.0, 4000.0),
                min_pep_length:int=5, max_pep_length:int=35)

*Lightweight digest that returns only peptide sequences as a set.

This is a memory-efficient alternative to digest() when only the peptide sequences are needed, without positional or mass annotations.*

Type Default Details
sequence str Protein sequence to digest
enzyme str trypsin Enzyme name (default: ‘trypsin’)
missed_cleavages int 0 Number of allowed missed cleavages (default: 0)
mass_range tuple (800.0, 4000.0) (min, max) monoisotopic mass filter in Da (default: (800.0, 4000.0))
min_pep_length int 5 Minimum peptide length to retain (default: 5)
max_pep_length int 35 Maximum peptide length to retain (default: 35)
Returns set Set of peptide sequences passing the length and mass filters

 fasta_to_peptide_set (fasta_path:Union[str,pathlib.Path],
                       enzyme:str='trypsin', missed_cleavages:int=0,
                       mass_range:tuple[float,float]=(800.0, 4000.0),
                       min_pep_length:int=5, max_pep_length:int=35,
                       show_progress:bool=True)

*Parse a FASTA file and return all unique peptides as a set.

Memory-efficient function that digests proteins on-the-fly without storing the full protein sequences. Useful for building canonical peptide reference sets for comparison with experimental data.*

Type Default Details
fasta_path Union Path to the FASTA file to parse
enzyme str trypsin Enzyme name (default: ‘trypsin’)
missed_cleavages int 0 Number of allowed missed cleavages (default: 0)
mass_range tuple (800.0, 4000.0) (min, max) monoisotopic mass filter in Da (default: (800.0, 4000.0))
min_pep_length int 5 Minimum peptide length to retain (default: 5)
max_pep_length int 35 Maximum peptide length to retain (default: 35)
show_progress bool True Show progress bar with tqdm (default: True)
Returns set Set of all unique peptide sequences from the FASTA file
